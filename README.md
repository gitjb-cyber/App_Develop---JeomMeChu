# App:점메추(Food Recommendations App)


## 소개

카테고리 선택 형식의 메뉴 추천 앱입니다. 
매운 음식과 맵지 않은 음식, 뜨거운 음식과 차가운 음식 등의 당일 원하는 느낌의 카테고리를 선택하면
해당 음식들을 한식/일식/중식/양식/아시안 순서로 나열됩니다.
